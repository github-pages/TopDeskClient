# Get-ProcessingStatus

## SYNOPSIS
Get list of processing statuses

## SYNTAX

```
Get-ProcessingStatus [<CommonParameters>]
```

## DESCRIPTION
This command will return the list of processing statuses name, id, onHold, processingState, response and default.

## EXAMPLES

### EXAMPLE 1
```
Get-ProcessingStatus
```
